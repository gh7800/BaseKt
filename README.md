#### a、发布tag方法
##### 1、git add → commit → push  提交代码
##### 2、git tag 1.0.0   // 添加tag
##### 3、git push origin 1.0.0  //提交tag

#### b、删除tag方法
##### 1. 删除本地tag  git tag -d tagName
##### 2. 删除远程tag  git push origin :refs/tags/tagName
